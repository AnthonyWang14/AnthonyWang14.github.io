# Hao Wang Academic Website

Source for the personal academic website at <https://AnthonyWang14.github.io>.

## Main Content

* homepage: `_pages/about.md`
* publications: `_pages/publications.md`
* teaching: `_pages/teaching.html`
* CV downloads: `files/`

## Local Development

1. Install Ruby, Bundler, and the project dependencies.
2. Run `bundle exec jekyll serve`.
3. Open `http://localhost:4000`.
